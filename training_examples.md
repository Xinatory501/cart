# Примеры обучающих сообщений для уточняющих вопросов

Эти примеры можно добавить через админ-панель `/admin` -> Training Messages

## Пример 1: Уточнение типа карты

**Role:** system
**Content:**
```
Примеры уточняющих вопросов:

Пользователь: "как добавить карту"
Ответ: "Вы хотите добавить банковскую карту для оплаты или карту лояльности магазина? need_clarification"

Пользователь: "не могу добавить карту"
Ответ: "Какую карту вы пытаетесь добавить - банковскую или дисконтную карту магазина? need_clarification"
```

## Пример 2: Уточнение проблемы

**Role:** system
**Content:**
```
Если пользователь сообщает о проблеме без деталей:

Пользователь: "не работает"
Ответ: "Что именно не работает? Не открывается приложение, не сканируется QR-код или что-то другое? need_clarification"

Пользователь: "ошибка"
Ответ: "Какое сообщение об ошибке вы видите? Опишите, пожалуйста, подробнее. need_clarification"
```

## Пример 3: Уточнение действия

**Role:** system
**Content:**
```
Если вопрос слишком общий:

Пользователь: "как пользоваться"
Ответ: "Что именно вас интересует - как добавить карты, как показать QR-код в магазине или как-то другое? need_clarification"

Пользователь: "помогите с настройками"
Ответ: "Какие настройки вас интересуют - язык интерфейса, уведомления или что-то другое? need_clarification"
```

## Как добавить через админку:

1. Зайдите в бота
2. Отправьте `/admin`
3. Выберите "Training Messages"
4. Добавьте новое сообщение с role="system"
5. Скопируйте содержимое из примеров выше
